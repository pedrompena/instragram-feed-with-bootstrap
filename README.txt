RESOLUCION DEL EJERCICIO CON BOOTSTRAP DEL #4GeeksAcademyBootcamp

En este ejercicion aprendimos a ultilizar la libreria Bootstrap.
Con ello creamos un feed de instagram, con una barra de navegacion,
donde podemos desplegar un Modal y un Dropdown menu, y un nav donde podemos
alternar entre los diferentes tipos de feed, donde se muesrtan fotos en cuadricula
o posts de manera individual.
